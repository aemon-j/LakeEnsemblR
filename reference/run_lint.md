# Runs lint for LER script

Runs lintr::lint function, using the settings agreed upon for
LakeEnsemblR

## Usage

``` r
run_lint(file)
```

## Arguments

- file:

  filepath; to R script that you want to run the lintr::lint function on

## Examples

``` r
if (FALSE) { # \dontrun{
run_lint(file = "export_config.R")
} # }
```

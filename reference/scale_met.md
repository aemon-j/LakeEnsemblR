# Format meteorological data for each model

Format dataframe into shape for the specified model

Scale meteorological parameters.

## Usage

``` r
format_met(met, model, config_file, folder = ".")

scale_met(met, pars, model, out_file = NULL)
```

## Arguments

- met:

  dataframe; in the format for that particular model with the column
  headers

- model:

  character; Model for which scaling parameters will be applied. Options
  include c("GOTM", "GLM", "Simstrat", "FLake", "MyLake")

- config_file:

  filepath; To LER config yaml file. Only used if model = "GOTM"

- folder:

  filepath; to folder which contains the model folders generated by
  export_config()

- pars:

  dataframe; scaling factors for meteorological data with column names
  corresponding to variable dictionary

- out_file:

  filepath; For scaled meteorlogical data

## Value

dataframe of met data in the model format

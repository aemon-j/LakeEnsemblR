# Export LakeEnsemblR standardized input to model specific driver format

Export driver files for each model

## Usage

``` r
export_meteo(
  config_file,
  model = c("GOTM", "GLM", "Simstrat", "FLake", "MyLake"),
  folder = "."
)
```

## Arguments

- config_file:

  filepath; to LakeEnsemblr yaml master config file

- model:

  vector; model to export driving data. Options include c('GOTM', 'GLM',
  'Simstrat', 'FLake')

- folder:

  filepath; to folder which contains the model folders generated by
  export_config()

## Examples

``` r
if (FALSE) { # \dontrun{
export_meteo(model = c('GOTM', 'GLM', 'Simstrat', 'FLake'),
             meteo_file = 'LakeEnsemblR_meteo_standard.csv')
```

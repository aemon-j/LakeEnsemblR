# Export initial conditions into each model setup from the LakeEnsemblR standardized input for observed temperature profile

Export initial condition files and input into model configuration files.

## Usage

``` r
export_init_cond(
  config_file,
  model = c("GOTM", "GLM", "Simstrat", "FLake", "MyLake"),
  date = NULL,
  print = TRUE,
  folder = "."
)
```

## Arguments

- config_file:

  filepath; to LakeEnsemblr yaml master config file

- model:

  vector; model to export driving data. Options include c('GOTM', 'GLM',
  'Simstrat', 'FLake')

- date:

  character; Date in "YYYY-mm-dd HH:MM:SS" format to extract the initial
  profile. If NULL, the observations file specified in config_file is
  used to extract the date.

- print:

  logical; Prints the temperature profile to the console

- folder:

  filepath; to folder which contains the model folders generated by
  export_config()
